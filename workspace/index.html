Schedule

작전명 : Duck 소굴로

조직 ‘GuGu’ 는 조직 ‘Duck’ 의 Boss인 도널드를 감시하기 위해 문주스피노박사에게 소형로봇을 만들 것을 명령한다. 문주스피노와 인규 이푸스는 소형로봇을 만들게 되고 로봇의 코드네임을 ‘Sphero’로 정하게 된다. 한달 후 ‘Sphero’는 본격적인 임무를 수행하게 된다.

#1 - 첫번째 보스의 지령

Boss의 지령 : Duck 조직의 근거지로 이동하라.

Duck 조직은 아무도 모르는 섬에 살고있다. 그들은 물 위를 움직일 수 있기 때문에 섬의 위치는 어쩌면 그 들에게는 당연한 위치 선정일지도 모른다. 우리의 Sphero 들은 이 험난한 물길을 뚫고 Duck 조직의 근거지로 이동해야 한다.

--동영상--

#2 - 근거지 도착

Boss의 지령 : 조직 내부로 침입해라.

Spider 조직에 들어가기 위해서는 그들만의 암호를 풀어야 한다, 단 10초 안에 { ROHDQ4X4 암호를 해독하시오, 10초안에 해독하지 못한다면 침입자로 간주합니다.}

--동영상--

#3- 내부 침입 1단계 (미로통과)

Boss의 지령 : Spider 보스의 사무실로 들어가라.

Duck 조직의 Boss인 도널드는 굉장히 겁쟁이다. 그래서 출입문 부터 자신의 집무실까지 미로로 복잡하게 설계해두었다. SP1과 SP2 는 만능키 문주스피노의 정보와 인규이푸스의 프로그래밍으로 인해 특별한 일이 없다면 무사히 통과 할 예정이다.

--동영상--

#4- Spider 보스 사무실 앞 문지기

Boss의 지령 : Duck 보스의 사무실로 들어가라. (MISSION 진행중)

예상대로라면, 무사히 통과가 가능했지만, Duck 조직의 라이언 보스가 예전의 쇠약해 보이던 문지기를 덩치가 있는 문지기로 교체하면서 생각과는 다르게 전개된다.

--동영상--

#5 - Spider 보스 방 침입

Boss의 지령 : Duck 보스의 사무실로 들어가라. (MISSION 진행중)

예상대로라면, 무사히 통과가 가능했지만, Duck 조직의 라이언 보스가 예전의 쇠약해 보이던 문지기를 덩치가 있는 문지기로 교체하면서 생각과는 다르게 전개된다.
